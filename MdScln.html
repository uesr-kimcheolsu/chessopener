흑은 d폰을 1칸 올려 밝은 색 비숍의 대각선을 엽니다. 오픈 시실리안으로 진행하면 일반적으로 마록지 바인드(5.c4)를 예방하기 위해 Nf6를 두고, 백은 Nc3로 수비합니다.(간혹 f3의 시실리안 프린스를 선택) 이후 흑은 여러 선택권을 갖는데:

  • Nf6(Nc3) + a6 = 나이도프(Najdorf)
  • Nf6(Nc3) + Nc6 = 클래시컬(Classical)
  • Nf6(Nc3) + e6 = 스헤버닝언(Scheveningen)
  • Nf6(Nc3) + e5 = 오픈-: 베니스 어택(Open-: Venice Attack)
  • Nf6(Nc3) + g6 = 드래곤(Dragon)
  • Nf6(Nc3) + Bd7 = 쿠프레치크(Kupreichik)
  • a6 = 이름 없음
  • Nc6 = 이름 없음
  • g6 = 모던 액셀러레이티드 드래곤(Modern Accelerated Dragon)
물론 백이 선택하는 안티 시실리안도 있습니다.
